# Dataset

This folder stores the raw dataset used in the YES Bank stock price analysis project.

## File

- `raw/yes_bank_stock_prices.csv`

## Columns

- `Date`: Month and year of the stock price record.
- `Open`: Opening stock price for the month.
- `High`: Highest stock price for the month.
- `Low`: Lowest stock price for the month.
- `Close`: Closing stock price for the month.

## Summary

- Rows: 185
- Columns: 5
- Period: Jul-2005 to Nov-2020
- Missing values: 0
- Duplicate rows: 0

The target variable for machine learning is `Close`.
